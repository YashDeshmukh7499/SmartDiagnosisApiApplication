# SmartDiagnosisApiApplication
## AI Integration Approach

The project uses a hybrid diagnosis strategy:

1. Rule-based symptom matching identifies likely medical conditions based on predefined symptom combinations.

2. AI-style structured response generation improves consistency by formatting each diagnosis into:

   * probable condition
   * probability percentage
   * recommended next medical action

This approach ensures predictable backend behavior while keeping output meaningful and extensible for future integration with external AI APIs such as OpenAI.
